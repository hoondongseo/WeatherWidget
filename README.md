# Weather Widget
## 프로젝트 소개
알아보고 싶은 도시나 나라의 이름을 입력한 후, 검색하면 현재 기온, 습도, 풍속이 출력되는 간단한 날씨 위젯입니다.

getLocation을 활용해서 사용자의 위치를 기반으로 날씨를 출력하는 기능을 추가해서 편의성을 높였습니다.
